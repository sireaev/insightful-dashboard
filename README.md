# InsightfulDashboard


### Getting started
```
cd insightful-dashboard
npm i
```

### Run mock API with 1k shifts and 10 employees
```
npx json-server seed.js
```

### Start the project
```
npm start
```

### Shutdown the project

```
Tap CTRL + C
```
